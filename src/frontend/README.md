# Portfolio - Internet Computer Deployment

This is a personal portfolio website built with React, TypeScript, Tailwind CSS, and deployed on the Internet Computer.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Smooth Animations**: CSS-based animations for elegant transitions
- **Type Animation**: Dynamic typing effect for role titles
- **Section Navigation**: Fixed navbar with smooth scrolling to sections
- **Resume Download**: Direct download link for PDF resume
- **Contact Links**: GitHub, LinkedIn, and email integration

## Local Development

### Prerequisites

- Node.js (v18 or higher)
- pnpm (or npm/yarn)
- DFX (Internet Computer SDK)
